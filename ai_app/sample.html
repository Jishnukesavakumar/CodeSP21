PL/SQL Error Codes

PL/SQL (Procedural Language/Structured Query Language) is Oracle Corporation's procedural extension for SQL and the Oracle relational database. Below are some common PL/SQL error codes and their descriptions:

Error Code Description
ORA-00001 Unique constraint violated.
ORA-00904 Invalid column name.
ORA-01017 Invalid username/password; logon denied.
ORA-01400 Cannot insert NULL into a column that does not allow NULLs.
ORA-01722 Invalid number.
ORA-06502 PL/SQL: numeric or value error.
ORA-06512 At line number (used for stack trace).
ORA-12154 TNS: could not resolve the connect identifier specified.
